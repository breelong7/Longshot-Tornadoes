About the team.

  • The main purpose of BuzzedFeed is to be a one stop shop for thirsty users.
  • The quiz will generate a drink option for a user based on preference.
  • The page will check local storage to see if the user had been to the page before and generate a custom greeting.
  • The recipes page will allow the user to find the ingredients of their favorite drinks.
  • The game will allow the user to play an interactive quiz-style trivia while drinking their favorite drink.
  • The entire experience will be interactive and entertaining & easy to navigate for the user.
  • The About Us page will show the user more information about team Longshot Tornadoes.
